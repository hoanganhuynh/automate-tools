# Automate tool
Automate tool for CodersX Content team

## Installtion
```
$ git clone https://github.com/nmtoan251998/automate-tool.git
$ cd automate-tool
$ npm install
```

## Available scripts
*Start the tool*
``` 
$ npm start
```

*Use browserify npm package to convert clientside JS code to user **require()** syntax*
```
$ npm run compile-script
```

## Dependencies
1. html-to-image
2. downloadjs
3. browsersify
4. JSONPath
5. date-time
6. express
